What's new ?

This release is mostly a bug-fix release...

New features

  • Added 'Import...' feature into project navigator context menu: this allows importing local files into the workspace.
  • Edit history is not cleared anymore on file save.
  • New welcome window on first lanunch.

Fixes

  • Display glitches in project navigator.
  • On first start when the welcome window is closed, it is still listed in the 'Windows' menu.
  • Children windows not closed on application exit.
  • Code editor: when paste on selected text, selected text should be replaced by pasted one.
  • HTML Preview and Remote HTML Preview cannot display pages when file path contains spaces.
  • Duplicate entries appearing in device list of HTML Preview.
  • Opened preview windows should not get focus after any action in project navigator.
  • Error message when closing last editor tab and a preview windows is opened.
