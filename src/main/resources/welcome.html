Welcome to Shift !

Shift is a code editor designed for the modern web. The perfect tool for your responsive and mobile web sites.

Features

  • Fast loading editor
  • Syntax coloration and completion for common web file formats
  • Project wizards to help you get started with your projects
  • Live preview and live remote preview

This version of Shift is a SNAPSHOT release. It maybe unstable.

If you encounter issues after upgrading from a previous version, please consider deleting your previous settings (.shift folder in user directory).
